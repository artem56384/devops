# Log Parser Script

## Описание

Скрипт `log_parser.sh` анализирует системный лог `/var/log/syslog` или `/var/log/messages` за последние 10 минут.

Скрипт ищет строки, содержащие слова `error`, `fail`, `critical` без учета регистра.

Найденные строки сохраняются в файл вида `errors_YYYY-MM-DD.log`.

После этого содержимое файла отправляется через POST запрос на `https://httpbin.org/post` с помощью `curl`.

## Обработка ошибок

1. Если лог-файл отсутствует, скрипт выводит сообщение об ошибке и завершает работу с кодом `1`.

2. Если отсутствует интернет или запрос не выполнен, скрипт делает повторную попытку через 10 секунд.

3. Максимальное количество попыток отправки — 3.

## Требования

Для работы скрипта должны быть установлены:
`bash`
`curl`
