# Load Average

## Что это такое

Load Average — это показатель загрузки системы, отражающий количество процессов:

- выполняющихся (running)
- ожидающих выполнения (в очереди на CPU)
- ожидающих ресурсы (например, I/O)

Это не просто загрузка CPU.

## Как рассчитывается

Load Average рассчитывается как экспоненциально взвешенное скользящее среднее (EWMA), а не простое арифметическое среднее.

Это означает:
- недавние значения имеют больший вес
- более старые значения постепенно теряют влияние

## Интервалы

Отображается за:
- 1 минуту
- 5 минут
- 15 минут

## Как интерпретировать

Сравнивается с количеством ядер CPU:

- значение ≈ количеству ядер → нормальная загрузка
- меньше → система справляется
- больше → есть очередь процессов

Важно:
Load Average учитывает не только CPU, но и процессы, ожидающие I/O.

## Команды для просмотра

uptime  
top  
cat /proc/loadavg  

## Вывод

Load Average — это не просто средняя нагрузка, а показатель количества процессов в системе, рассчитанный через EWMA.
