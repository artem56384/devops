# Права доступа в Linux

## Категории

- User (владелец)
- Group (группа)
- Others (остальные)

## Типы прав

- r — чтение
- w — запись
- x — выполнение

## Просмотр прав

ls -l

## Изменение прав (chmod)

chmod используется для изменения прав доступа.

## Числовое представление прав

Права доступа могут задаваться в числовом (восьмеричном) виде.

Каждому типу прав соответствует число:

- 4 — read (чтение)
- 2 — write (запись)
- 1 — execute (выполнение)

Суммируя значения, получаем права:

- 7 = 4 + 2 + 1 → rwx
- 6 = 4 + 2 → rw-
- 5 = 4 + 1 → r-x
- 4 = 4 → r--
- 0 → ---

Пример:

chmod 755 file

Расшифровка:
- 7 (user) → rwx
- 5 (group) → r-x
- 5 (others) → r-x

chmod 644 file

- 6 → rw-
- 4 → r--
- 4 → r--

## Изменение владельца (chown)

chown user file

Изменить владельца и группу:

chown user:group file

## Изменение группы (chgrp)

chgrp group file
