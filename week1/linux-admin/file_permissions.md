# Права доступа к файлам в Linux

## Категории прав

В Linux права доступа разделяются на три категории пользователей:

1. User  
Это владелец файла

2. Group  
Это группа, к которой относится файл

3. Others  
Это все остальные пользователи

## Основные виды прав

Существует три основных права:

1. `r`  
Read, право на чтение

2. `w`  
Write, право на запись

3. `x`  
Execute, право на выполнение
