# Bash

## Краткое описание

Bash (Bourne Again Shell) — это командная оболочка и язык сценариев в операционных системах семейства Linux.
Она используется для выполнения команд, управления системой и автоматизации задач с помощью скриптов.

Bash является стандартной оболочкой во многих дистрибутивах Linux и широко применяется в администрировании.

## Что такое bash-скрипт

Bash-скрипт — это текстовый файл, содержащий последовательность команд, которые выполняются интерпретатором Bash.

Скрипты позволяют автоматизировать выполнение задач и объединять несколько команд в один исполняемый файл.

## Требования к bash-скрипту

Чтобы файл мог выполняться как bash-скрипт, должны быть выполнены следующие условия:

### 1. Указание интерпретатора

В первой строке файла должен быть указан интерпретатор, который будет выполнять команды.
Это делается с помощью специальной конструкции, называемой shebang.

### 2. Права на выполнение

Файл должен иметь права на выполнение. Без этого система не позволит запустить его как программу.

### 3. Корректный синтаксис

Скрипт должен быть написан без синтаксических ошибок и содержать корректные команды Bash.

### 4. Формат файла

Файл должен быть обычным текстовым файлом без лишних символов и с корректной кодировкой.

### 5. Способ запуска

Скрипт должен запускаться через оболочку или напрямую как исполняемый файл.
