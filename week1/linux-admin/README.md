# Week 1

## Описание

В данной директории находятся материалы по заданиям первой недели курса DevOps.

## Структура

`linux-admin/`  
Материалы по администрированию Linux

`networking/`  
Материалы по сетевым заданиям

`scripts/`  
Скрипты для автоматизации

`gitlab-mr-link.txt`  
Файл для ссылки на merge request

## Содержимое linux-admin

1. `systemd.md`  
Описание systemd, директории хранения unit-файлов и основные команды

2. `load_average.md`  
Описание Load Average и способы интерпретации нагрузки

3. `file_permissions.md`  
Описание категорий прав на файлы и основных команд изменения прав

4. `linux_signals.md`  
Описание сигналов Linux, их типов и основных сигналов
