# Git

## Краткое описание

Git — это распределённая система контроля версий, предназначенная для отслеживания изменений в файлах и совместной работы над проектами.

Git позволяет сохранять историю изменений, работать с ветками, объединять изменения и откатываться к предыдущим версиям.

## Основные понятия

Репозиторий — это хранилище проекта и его истории изменений.

Коммит — это сохранённое состояние проекта в определённый момент времени.

Ветка — это отдельная линия разработки.

Удалённый репозиторий — это версия проекта, хранящаяся на сервере.

# Основные команды

## Инициализация

```bash
git init
```

## Проверка состояния

```bash
git status
```

## Добавление файлов

```bash
git add <file>
git add .
```

## Коммит

```bash
git commit -m "message"
```

## История

```bash
git log
```

## Ветки

```bash
git branch <name>
git checkout <name>
git checkout -b <name>
```

## Слияние

```bash
git merge <branch>
```

## Удалённый репозиторий

```bash
git remote add origin <url>
git push
git pull
```
